===================== Chinese ==============================
ViewPage 能做些什么：
    ViewPage插件用于分析当前浏览的网页，它所能分析的内容有：
    页面基本信息：(编码，网页尺寸，Cookies ...)
    框架结构
    图像
    源代码
    链接
    Flash 动画
    CSS 样式表
    VB/JS 脚本
    媒体挖掘
    W3C验证/域名信息查询/RSS 验证

如何使用ViewPage插件
    点击"外部工具栏上"的ViewPage按钮即可启动ViewPage分析当前
    页，分析方式分为两种：
      1、整页分析：分析整个页面；
      2、选中分析：选中网页中的某一部分后，运行插件
        这时，ViewPage 会将你所选择的部分在一个新窗口中打开
	然后再对该新窗口进行分析
    运行插件后，你会发现新开了一个标题为"ViewPage ..."的窗口
    切换到该窗口，点击相应标签，你就可以看到分析的结果了。

注意：
    如果页面为框架结构，某些资源可能不能正常分析出来，这时你
    只要切换到"框架"标签，点击相应框架的链接地址，打开该框架
    为一个单独的页面，再对其运行ViewPage分析就可以了。

===================== English ==============================
What ViewPage can do for you :
    ViewPage will allow you to view different resources in 
    a new page - images, Flash, links, media files, css, js,
    etc.

How to use ViewPage :
    2 Ways:
    1, Select part of page that you want to analyse,and run
       ViewPage.
       ViewPage just analyse your Selection;
    2, No Selection:
       ViewPage will analyse full page;

Note:
    If there are frames in page, Maybe ViewPage can't get 
    resources rightly, go to "Frame" tab, cilck the link
    of frame to open it in new window, and then run ViewPage
    again.
